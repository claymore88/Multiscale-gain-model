# Multiscale-gain-model
Code of coupled gain-rate-equations for GHz fiber laser
PQSML M files are the codes for the models without gain perturbation
Two gain rate equations are combined with the NLSE to constitute the model
Two cases, i.e., repetition rates of 10GHz and 20GHz are given

In the files named as 'With perburbation'
Perburbed form of the gain rate equation is employed
The code is specified to the case of 4.6GHz mode-locking
